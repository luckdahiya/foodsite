CCNA Course Syllabus

Module 1: Networking Fundamentals

  • Introduction to Networking
    • Network types and topologies
    • OSI and TCP/IP models
  • Networking Devices
    • Routers, switches, and hubs
    • Network cables and connectors

Module 2: IP Addressing and Subnetting

  • IP Addressing
    • IPv4 and IPv6 addressing
    • Private and public IP addresses
  • Subnetting
    • Subnet masks and CIDR notation
    • Subnetting IPv4 addresses

Module 3: Routing and Switching

  • Routing Basics
    • Static and dynamic routing
    • Routing protocols: RIPv2, OSPF
  • Switching Concepts
    • VLANs and trunking
    • Spanning Tree Protocol (STP)

Module 4: IP Services

  • Network Address Translation (NAT)
    • Understanding NAT and PAT
    • Configuring NAT
  • Dynamic Host Configuration Protocol (DHCP)
    • DHCP operation and configuration
    • DHCP troubleshooting

Module 5: Security Fundamentals

  • Network Security
    • Basic security concepts
    • Implementing access control lists (ACLs)
  • Securing Devices
    • Securing routers and switches
    • Security best practices

Module 6: Automation and Programmability

  • Network Automation
    • Introduction to network automation
    • Basic concepts of network programmability
  • Network Management Tools
    • Understanding network management tools
    • Basic configuration and monitoring
