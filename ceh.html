CEH Course Syllabus for GICSEH

Explore the comprehensive syllabus for the Certified Ethical Hacking (CEH) course at GICSEH in Preet Vihar, Delhi.

CEH Course Syllabus

Module 1: Introduction to Ethical Hacking

  • Overview of Ethical Hacking
    • Ethics and Legality
    • Cybercrime and its Impact
  • Understanding Hacker Mindset
    • Types of Hackers
    • Five Phases of Hacking

Module 2: Footprinting and Reconnaissance

  • Information Gathering Techniques
    • Footprinting Concepts and Methodologies
    • Tools for Footprinting
  • Active and Passive Reconnaissance
    • OSINT (Open Source Intelligence)
    • Whois, DNS, and Network Queries

Module 3: Scanning Networks

  • Network Scanning Techniques
    • Types of Scans (TCP, SYN, etc.)
    • Banner Grabbing and OS Fingerprinting
  • Vulnerability Scanning
    • Using Tools like Nmap, Nessus
    • Identifying Network Vulnerabilities

Module 4: Enumeration

  • Enumeration Techniques
    • NetBIOS, SNMP, LDAP Enumeration
    • Enumerating Users and Groups
  • System Hacking
    • Gaining Access to Systems
    • Privilege Escalation Techniques

Module 5: Vulnerability Analysis

  • Understanding Vulnerabilities
    • Types of Vulnerabilities
    • Common Vulnerability Scoring System (CVSS)
  • Using Vulnerability Assessment Tools
    • Vulnerability Scanning Tools
    • Analyzing and Reporting Vulnerabilities

Module 6: Malware Threats

  • Types of Malware
    • Viruses, Worms, Trojans, and Ransomware
    • Understanding Rootkits and Keyloggers
  • Analyzing and Detecting Malware
    • Anti-malware Tools and Techniques
    • Malware Obfuscation and Encryption

Module 7: Sniffing and Evasion

  • Sniffing Concepts
    • Understanding Packet Sniffing
    • Tools like Wireshark and Tcpdump
  • Evasion Techniques
    • IDS, IPS, Firewall Evasion Techniques
    • MAC and IP Spoofing

Module 8: Social Engineering

  • Social Engineering Concepts
    • Understanding Human Vulnerabilities
    • Types of Social Engineering Attacks
  • Social Engineering Techniques
    • Phishing, Pretexting, Baiting, and Tailgating
    • Countermeasures Against Social Engineering

Module 9: Denial of Service (DoS) and DDoS Attacks

  • DoS and DDoS Concepts
    • Understanding DoS Attacks
    • Techniques for Launching DoS Attacks
  • Defensive Techniques
    • DoS and DDoS Prevention
    • DoS Attack Detection and Mitigation

Module 10: Session Hijacking

  • Session Hijacking Techniques
    • Understanding Session Hijacking
    • Types of Session Hijacking (Active, Passive)
  • Countermeasures
    • Session Hijacking Prevention
    • Using Secure Communication Channels

Module 11: Hacking Web Servers and Applications

  • Web Server Attacks
    • Understanding Web Server Vulnerabilities
    • Types of Attacks: Directory Traversal, Command Injection
  • Web Application Attacks
    • Cross-site Scripting (XSS)
    • SQL Injection
    • Countermeasures and Security Best Practices

Module 12: Wireless Network Hacking

  • Wireless Network Concepts
    • Understanding Wireless Networks (WEP, WPA, WPA2)
    • Common Wireless Network Vulnerabilities
  • Wireless Hacking Techniques
    • Cracking WEP/WPA2 Encryption
    • Using Tools like Aircrack-ng

Module 13: Cryptography

  • Introduction to Cryptography
    • Understanding Encryption and Decryption
    • Types of Cryptographic Algorithms
  • Cryptographic Attacks
    • Breaking Encryption with Brute Force
    • Countermeasures and Best Practices

Module 14: Cloud Computing and IoT Hacking

  • Cloud Computing Security
    • Understanding Cloud Computing Models
    • Security Concerns and Best Practices
  • IoT (Internet of Things) Security
    • Introduction to IoT Security
    • Common Vulnerabilities in IoT Devices

Module 15: Penetration Testing and Security Audit

  • Penetration Testing Methodologies
    • Types of Penetration Testing (Black-box, White-box)
    • Penetration Testing Tools and Frameworks
  • Security Audit Techniques
    • Conducting a Security Audit
    • Reporting Vulnerabilities and Remediation
